## What Made This Possible

| Feature | Gameplay Impact |
|---------|-----------------|
| **Temperature** | Morning chill, altitude changes, storm cooling |
| **Weather** | Fog reducing visibility, rain affecting combat/tracks |
| **Flora** | Harvestable berries, environmental description |
| **Fauna** | Deer grazing at dawn, wolves hunting at dusk, owl waking at night |
| **Activity Periods** | Deer (diurnal) gone at night, owl/wolves (nocturnal) appear |
| **Spawn Conditions** | Wolves don't spawn in storms, deer avoid predator areas |
| **Tracks** | Reading animal movement, rain washing away evidence |
| **Roads** | Safety from wildlife, faster travel, navigation points |
| **Biome Coherence** | Mountain = cooler, pine trees, eagles; Forest = oaks, deer, wolves |
